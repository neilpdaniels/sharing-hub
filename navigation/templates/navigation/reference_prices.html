SharingHub currently uses the LMBA fixing prices. Each metal's prices are generated at set times dependant on the precious metal in question. See individual metals below for a more information. Prices will be taken in USD and converted to GBP using the current FX spot price (the current conversion rate).

There will be a short delay from the official publishing until we receive the prices and apply to the relevant orders; this delay is monitored and controlled. SharingHub holds no responsibilty for any losses incurred due to incorrectly priced orders, though any erroroneously priced errors can be cancelled without consent of the other party, by raising a dispute request on the transaction.

Wastaco does not currently offer a reference price that uses the current bid/offer prices and so any prices should be set allowing for the expected variation in prices between fixing times.

Should you be interested in pricing your SharingHub orders independently via our secure API, else if you require more information on our refernce prices, please contact us

Current Gold Prices

The LBMA Gold price is set twice daily at 10:30 and 15:00 in an auction independently operated and administered by ICE Benchmark Administration. The price is set in US dollars per fine troy ounce. The price is set for that current point in time, and will be the base price for many large transactions. This process only runs every business day, with exceptions on Christmas eve and New Year's eve, where it will run only once at 10:30 AM.

Current Silver Prices

The LBMA Silver price auction is independently operated and administered by ICE Benchmark Administration.
The price is set daily in US dollars per ounce at 12:00 noon London time, on all business days
